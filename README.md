### Install gmt

```shell
go get -u github.com/2637309949/gmt/comm/gmt
```

```
Code generation tool for go-micro

Usage:
  gmt [command]

Available Commands:
  help        Help about any command
  init        Init a micro collection
  new         New service

Flags:
  -h, --help   help for gmt

Use "gmt [command] --help" for more information about a command.
```
